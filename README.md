# Valor Futuro (VF)
```
Vfuturo = VF(A = 500, r = 0.05706, n = 10)
```

# Anualidad (A)
 ```
Anualidad = A(VF = 6500, r = 0.05706, n = 10)
```

# Número de Pagos (n)
 ```
NPagos = n(VF = 6500, r = 0.05706, A = 500)
```

# tasa de interes (r) VF
```
tasa = round(r(VF = 6500, A = 500, n = 10),4)
```
